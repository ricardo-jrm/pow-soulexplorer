### <b>Usage</b>

```tsx
// install deps
yarn install

// run dev server
yarn dev

// run lint
yarn lint

// run unit tests
yarn test

// run e2e tests
yarn cy:run

// run storybook
yarn sb

// build app
yarn build

// start app
yarn start

// export app
yarn export
```
